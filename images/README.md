# 批次套用說明圖片資料夾

請將以下三張圖片檔案放置在此資料夾中：

## 需要的圖片檔案：

1. **bar-mode.png** - Bar 模式說明圖片
   - 展示快速套用單個欄位內容的介面

2. **panel-mode.png** - 面板模式說明圖片
   - 展示可套用全部欄位內容的完整介面

3. **mode-comparison.png** - 功能差異比較圖片
   - 展示 Bar 模式與面板模式的功能差異

## 注意事項：

- 圖片格式建議使用 PNG 或 JPG
- 建議圖片尺寸：寬度 800px 以上，保持適當的解析度
- 檔案名稱必須完全符合上述命名規則
- 放置圖片後，重新整理網頁即可看到效果

如果沒有放置圖片，系統會自動顯示提示文字。
